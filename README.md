# Java Data Structures & Algorithms (DSA) Practice

Welcome to my **Java DSA Process** repository! 🚀 This repository contains my daily practice code, covering fundamental concepts, Data Structures, and Algorithms in Java. It serves as a personal log of my coding journey and a comprehensive reference for solving various computational problems.

## 📂 Repository Structure

The `src` folder is organized by topics and concepts:

- **`Array/` & `twoDArr/`** - 1D and 2D Array manipulations, traversing, and basic array algorithms.
- **`ArrayList/`** - Dynamic array implementations and operations.
- **`String/`** - String manipulation, parsing, and algorithmic string problems.
- **`pattern/` & `PATTERNS/`** - Logic building through printing various star and number patterns.
- **`bitManuplation/`** - Bitwise operations and optimizations.
- **`functopns/`** - Java methods and functional problem-solving.
- **`oops/`** - Object-Oriented Programming concepts (Classes, Objects, Inheritance, Polymorphism).
- **`recursion/` & `backtracking/`** - Recursive problem solving and exhaustive search algorithms.
- **`shorting/`** (Sorting) - Implementation of classic sorting algorithms (Bubble, Selection, Insertion, Merge, Quick, etc.).
- **`diviedandconquer/`** - Advanced problem solving using the Divide & Conquer paradigm.
- **Basics** - Basic loops (`for_loop.java`, `do_while.java`, `nestedLoop.java`) and control flow statements.

## 🛠️ Technologies Used
- **Language**: Java
- **IDE**: Visual Studio Code / IntelliJ IDEA

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Suraj-kummar/java_process.git
   ```
2. Navigate to the specific topic folder.
3. Compile and run the Java files using your preferred IDE or from the command line:
   ```bash
   javac Filename.java
   java Filename
   ```
